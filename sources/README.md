# Зафіксовані джерела

Цей каталог містить артефакти, потрібні для відтворюваної обробки
оригінальних документів із `raw/`.

Для ЄФВВ 2024 з інформаційних технологій:

```bash
node scripts/fix-yefvv-2024-source.mjs
```

Скрипт:

1. перевіряє SHA-256 оригінального PDF;
2. зчитує його технічні метадані;
3. рендерить усі сторінки у PNG з роздільністю 180 DPI;
4. перевіряє кількість і формат сторінок;
5. створює `sources/yefvv-it-2024/source.json` із метаданими, шляхами,
   розмірами та контрольними сумами всіх артефактів.

Оригінальний PDF у `raw/` скрипт не змінює.

Після фіксації джерела чернетковий текст і YAML можна створити командою:

```bash
node scripts/extract-yefvv-2024-draft.mjs
```

Екстрактор перевіряє checksum PDF і формує:

- `sources/yefvv-it-2024/text/` — очищений посторінковий текст;
- `data/drafts/yefvv-it-2024.yaml` і `.json` — 140 машинно витягнутих
  завдань;
- `reports/yefvv-it-2024-draft-extraction.json` — статистику й перелік
  завдань для ручної візуальної обробки.

Складний контент обробляється наступною командою:

```bash
node scripts/enrich-yefvv-2024-complex-content.mjs
```

Вона витягує та інвентаризує 45 вбудованих зображень, транскрибує формули,
код і таблицю, створює веб-assets та записує результат у
`data/enriched/yefvv-it-2024.yaml`.

Після обробки складного контенту весь набір нормалізується командою:

```bash
node scripts/normalize-yefvv-2024.mjs
```

Нормалізатор:

- прибирає версткові переноси рядків, не змінюючи зміст джерела;
- зберігає нумеровані списки та дефіси у складених словах;
- переводить SQL і текстові команди у типізовані `code`-блоки;
- перевіряє сім пакетів по 20 питань і всі завдання на межі сторінок;
- не змінює офіційні відповіді, посилання на джерело та складний контент;
- створює `data/normalized/yefvv-it-2024.yaml` і звіт
  `reports/yefvv-it-2024-normalization.json`;
- фіксує повну ручну звірку 29 сторінок, 140 питань і 54 пов'язаних
  медіаресурсів у `reports/yefvv-it-2024-manual-validation.json`.

Повна автоматична валідація запускається командою:

```bash
node scripts/validate-yefvv-2024.mjs
```

Валідатор перевіряє YAML та JSON, структуру всіх типів контент-блоків,
унікальність ідентифікаторів, варіанти й офіційні відповіді, сторінки
джерела, checksum PDF, файлові посилання та завершення ручної перевірки
візуальних завдань. Окрема перевірка гарантує, що всі 140 питань звірено
з рендерами відповідних сторінок і пов'язаними медіа. Формальна схема розташована у
`schemas/yefvv-it-question-dataset.schema.json`, а результат записується в
`reports/yefvv-it-2024-validation.json`.

Зіставлення з офіційною програмою 2025 року виконується командами:

```bash
node scripts/classify-yefvv-2024-program-2025.mjs
node scripts/validate-yefvv-2024-classification.mjs
```

Перший скрипт створює формальну таксономію з 10 розділів і 127 тематичних
позицій, класифікує всі 140 питань та записує:

- `data/taxonomies/yefvv-it-program-2025.yaml`;
- `data/classified/yefvv-it-2024.yaml`;
- `reports/yefvv-it-2024-program-2025-alignment.json`.

Другий скрипт перевіряє еквівалентність YAML/JSON, посилання на теми,
офіційні ваги та когнітивні рівні, checksum програми, незмінність
офіційного змісту і узгодженість звіту. Результат:
`reports/yefvv-it-2024-classification-validation.json`.

Пояснення до всіх варіантів відповідей створюються й перевіряються
командами:

```bash
node scripts/explain-yefvv-2024.mjs
node scripts/validate-yefvv-2024-explanations.mjs
```

Генератор не змінює офіційний ключ. Він додає до кожного питання короткий
навчальний висновок, feedback для чотирьох варіантів, статус незалежної
перевірки відповіді, тему програми та джерела. Результати:

- `data/explained/yefvv-it-2024.yaml`;
- `reports/yefvv-it-2024-explanations.json`;
- `reports/yefvv-it-2024-explanations-validation.json`.

Для питань 82 і 102 офіційний маркер збережено, але пояснення має статус
`disputed`, оскільки жоден варіант не узгоджується зі стандартним
предметним означенням.

Фінальний реліз для вебзастосунку збирається й перевіряється командами:

```bash
node scripts/assemble-yefvv-2024-release.mjs
node scripts/validate-yefvv-2024-release.mjs
```

Складальник створює:

- `data/yefvv-it-2024.yaml` як канонічний набір версії `1.0.0`;
- `data/yefvv-it-2024.json` як еквівалентне представлення для клієнта;
- `data/yefvv-it-2024.manifest.json` з інвентарем і SHA-256 усіх
  релізних ресурсів.

Фінальний валідатор виконує 15 перевірок і записує результат у
`reports/yefvv-it-2024-release-validation.json`. Він охоплює 140 питань,
560 варіантів, 9 вебресурсів, 45 вихідних зображень, офіційні PDF та
результати попередніх валідаторів.

`6480e97f0a8ae729b3d4fe41628c83e575f4d840e1ebd0ad1d99783f736a326e`.
