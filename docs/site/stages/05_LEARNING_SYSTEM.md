# Етап 5. Персональна система навчання

**Статус:** реалізовано.

## Ціль

На основі історії відповідей рекомендувати користувачеві корисне
тренування, а не випадковий набір уже знайомих питань.

## Роботи

- створити список помилок;
- додати закладки;
- обчислювати рівень засвоєння питання;
- реалізувати простий алгоритм інтервального повторення;
- формувати щоденну сесію;
- визначати слабкі теми;
- створити dashboard прогресу;
- показувати точність і час за темами та періодами;
- відрізняти першу відповідь від повторних спроб;
- додати цільову дату іспиту;
- реалізувати експорт, імпорт і очищення прогресу.

## Початкове правило повторення

| Результат | Наступне повторення |
|---|---|
| Неправильно | сьогодні або наступна сесія |
| Правильно з попередніми помилками | через 1 день |
| Двічі правильно | через 3 дні |
| Тричі правильно | через 7 днів |
| Стабільно правильно | через 14-30 днів |

Алгоритм зберігається як окрема чиста функція, щоб пізніше його можна
було змінити без міграції всього UI.

## Результат

Головна сторінка відповідає на питання «що мені вчити сьогодні?» і
показує вимірюваний прогрес за програмою.

## Реалізація

- щоденний план об'єднує помилки, пропуски, заплановані повторення,
  закладки, слабкі теми та нові питання;
- кожне рекомендоване питання містить пояснення причини вибору;
- рівень засвоєння і дата повторення обчислюються окремим планувальником;
- прогрес показує перші та повторні спроби, рівні засвоєння, статистику
  за 7 і 30 днів та за весь час;
- слабкою вважається тема з точністю нижче 70% після щонайменше двох
  наданих відповідей;
- час сесії для статистики за темами орієнтовно розподіляється між її
  питаннями порівну;
- налаштування містять цільову дату іспиту та розмір щоденної сесії;
- дані можна експортувати, імпортувати з підтвердженням або очистити;
- сховище автоматично мігрує локальні дані з формату етапу 4.

## Критерії приймання

- [x] щоденна сесія пояснює, чому вибрано кожне питання;
- [x] слабкі теми визначаються за мінімальною кількістю спроб;
- [x] повторна відповідь не переписує історію;
- [x] експортований файл можна імпортувати у чистий браузер;
- [x] очищення прогресу потребує явного підтвердження;
- [x] статистика коректно працює за відсутності спроб.
