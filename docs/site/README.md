# План розробки навчального сайту

## Мета

Створити статичний React-застосунок для підготовки до вступних іспитів у
магістратуру. Перша версія працює з ЄФВВ з інформаційних технологій, але
модель даних та інтерфейс мають підтримувати майбутнє додавання ЄВІ:
ТЗНК та іноземної мови.

Сайт має не лише показувати питання, а й допомагати користувачеві:

- проходити повні та тематичні тести;
- розуміти причини помилок;
- бачити слабкі теми;
- повторювати недостатньо засвоєні питання;
- стежити за прогресом до іспиту;
- переходити до нових згенерованих завдань після вивчення офіційних.

## Зафіксовані технологічні рішення

- UI: React і TypeScript;
- збірка: Vite;
- маршрутизація: React Router із hash-маршрутами для надійної роботи на
  GitHub Pages без серверного налаштування;
- постійні дані користувача: версіоноване сховище в `localStorage`;
- оформлення: світла, темна або системна колірна тема;
- контент тестів: статичні JSON-файли та медіаресурси;
- деплой: GitHub Pages через GitHub Actions;
- сервер, акаунти й хмарна синхронізація не потрібні для MVP.

## Спільне визначення готовності

Кожен етап вважається завершеним, коли:

1. Реалізовані всі обов'язкові функції етапу.
2. Пройдені автоматичні перевірки.
3. Основний сценарій перевірено в браузері на мобільному й широкому
   екранах.
4. Сайт успішно збирається з коректним шляхом для GitHub Pages.
5. Дані користувача після оновлення сторінки не пошкоджуються.
6. Документація відповідає фактичній реалізації.
